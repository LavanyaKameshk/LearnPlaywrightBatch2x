# Learn Playwright Batch 2x

This repository contains Playwright practice code for the Learn Playwright Batch 2x course.

## Structure

- `Chapter_01_Basics/`: Basic JavaScript and Playwright code examples.

## Files

- `Chapter_01_Basics/01_Basics.js`: Basic JavaScript output examples.
- `Chapter_01_Basics/identifierBasics.js`: Identifier naming examples with snake_case and PascalCase.

## Usage

Run JavaScript files with Node.js:

```bash
node Chapter_01_Basics/01_Basics.js
node Chapter_01_Basics/identifierBasics.js
```
